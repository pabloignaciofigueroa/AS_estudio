# Content And Conversion

## Editorial Rule

Do not try to explain everything in the hero.

Follow this sequence:

1. Create respect.
2. State the product.
3. Show proof quickly.
4. Explain capabilities.
5. Segment by audience.
6. Show ecosystem depth.
7. Ask for action.

## Content Lessons

- Keep the hero tight.
- Delay the longer explanation until after visual proof.
- Write by audience, not just by feature.
- Repeat CTA with different context.
- Use product language that feels precise and operational.

## Conversion Lessons

- Offer a primary action early.
- Offer a secondary action for less-ready users.
- Branch the conversion path if the audience has two buying modes.
- Reinforce the final CTA after enough proof.

## Blog And Freshness

Use a live content layer when the project benefits from:

- trust
- product velocity
- ecosystem depth
- SEO

Do not add a blog block if it will remain abandoned.

## Footer As Conversion Continuity

Treat the footer as:

- one more credibility surface
- a route map
- a brand reinforcement surface

Do not let the footer collapse into generic legal-only links.
