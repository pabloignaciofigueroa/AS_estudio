# Source And Stack

## Audit Scope

Analyze the local capture only.

Work from these files:

- `Google Antigravity.html`
- `Google Antigravity_files/styles-7KLEMMT6.css`

Treat the site as a compiled capture, not as the original repository.

## Confirmed Technical Findings

- Use `Angular 20.3.11`.
- Treat the site as an SPA with routed sections.
- Use `Google Sans Flex` as the main type family.
- Use `Google Sans Code` for technical accents.
- Use `Google Symbols` for iconographic text.
- Use `GSAP` for motion orchestration.
- Use `ScrollTrigger` for scroll-based choreography.
- Use drag and inertia in sliders.
- Use `PoissonDiskSampling` to distribute particles.
- Use `canvas`-based graphic layers for particle systems.
- Use GTM and cookie bar infrastructure.

## Treat As Likely But Not Fully Closed

- Treat advanced particle or morphing layers as confirmed visual systems.
- Do not claim direct `Three.js` usage unless the missing remote chunk is available.

## Bundle Reality

Read the site as production software, not as a static brochure:

That depth is part of why the site feels serious.
