# Visual System

## Typography

Use three coordinated families or equivalents:

- primary brand / product font
- technical accent font
- iconographic symbol font

Antigravity uses:

- `Google Sans Flex`
- `Google Sans Code`
- `Google Symbols`

## Typography Rules To Reuse

- Use a full type scale.
- Set explicit line-height for each level.
- Set explicit tracking for each level.
- Keep CTA typography separate from body typography.
- Keep headings weight-controlled and optically stable.

## Grid

Use a real system, not arbitrary containers:

- desktop: `12` columns
- tablet: `8` columns
- mobile: `4` columns

Use page margins that visibly change by breakpoint:

- desktop around `72px`
- tablet around `40px`
- mobile around `16px`

## Shape Language

Use a small set of radii only:

- very small
- small
- medium
- large
- xl
- rounded pill

Antigravity leans on large rounded corners for cards and pills for controls.

## Color

Build a neutral-heavy palette:

- dark structural grey
- mid grey for secondary text
- light surface greys
- one controlled accent color

Use the accent for:

- active states
- focus
- tabs
- highlighted navigation

Avoid turning every element into an accent moment.

## Blur And Overlays

Use blur with restraint on:

- header
- menus
- secondary buttons
- chips
- video overlays

Treat blur as a hierarchy and atmosphere tool, not as decoration.

## SVG And Premium Finish

Use custom SVG work when:

- the brand mark matters
- the footer should feel designed, not appended
- the visual finish needs depth through masks or filters

Do not fake this with random stock blobs.
